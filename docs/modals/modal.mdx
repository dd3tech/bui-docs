# <span name="floating-nav">Modal</span>

The Modal component in React allows you to create custom dialogs that grab the user's attention or provide important information. By displaying content in a
separate window that overlays the rest of the page, Modals can be a powerful tool for prompting user actions or displaying critical data. With the ability to
fully customize the content and appearance of your Modal, you can create a unique user experience that enhances the usability and effectiveness of your web
application.

##### <span name="floating-nav">Usage</span>

Use the Modal component as shown below:

<ContainerComponentDoc>
    <ModalCustom hasAButton messageButtons={['Open Modal']} />
</ContainerComponentDoc>

The code snippet below shows how to use the Modal component:

<WindowEditor
    codeString={`import { Modal } from 'dd360-ds''

<Modal active={true} setCloseModal={() => alert('function to close the modal')}>
    This is an example modal
</Modal>
`}/>

<br />

##### <span name="floating-nav">Animation</span>

With the prop <Tag text='animation' /> you can add an animation when opening the modal

<ContainerComponentDoc>
    <ModalCustom propsCustom={{ animation: true }} messageButtons={['Open Modal with animation', 'Open Modal whithout animation']} />
</ContainerComponentDoc>
