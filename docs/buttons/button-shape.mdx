# <span name="floating-nav">ShapeButton</span>

The ShapeButton component is a button that can be customized with different variants and optionally with a custom icon.

<br />
<br />
##### <span name="floating-nav">Import</span>

<WindowEditor codeString="import { ShapeButton } from 'dd360-ds'" />
<WindowEditor codeString="import { ShapeButton } from 'dd360-ds/Buttons'" />
<WindowEditor codeString="import ShapeButton from 'dd360-ds/ShapeButton'" />

<br />
<br />
##### <span name="floating-nav">Usage</span>

<br />
##### <span name="floating-nav">BaseCircleButton</span>

To use the BaseCircleButton follow this example:

<ContainerComponentDoc>
    <ShapeButton.BaseCircleButton />
</ContainerComponentDoc>

<WindowEditor codeString="import { ShapeButton } from 'dd360-ds'

<ShapeButton.BaseCircleButton />" />

<br />
##### <span name="floating-nav">Variants BaseCircleButton</span>

The prop variant can only be used with the BaseCircleButton, it has 2 variants: "square" and "circle". By default it is "circle".

<ContainerComponentDoc className="flex align-center items-center gap-x-3">
    <div className="flex flex-col align-center items-center">
        <Text className="flex mb-1">Square</Text>
        <ShapeButton.BaseCircleButton variant="square" />
    </div>
    <div className="flex flex-col align-center items-center">
        <Text className="flex mb-1">Circle</Text>
        <ShapeButton.BaseCircleButton variant="circle" />
    </div>
</ContainerComponentDoc>

<WindowEditor codeString='import { ShapeButton } from "dd360-ds"

<ShapeButton.BaseCircleButton variant="square" />
<ShapeButton.BaseCircleButton variant="circle" />' />
