# Input

    Inputs on websites are interactive elements that allow users to perform specific actions, such as submitting a form, navigating to another page, or making
    a purchase.

#### Variants

The "Input" comes with variants, by default it uses the "default" variant, but you can also use these: "active", "disabled", "warning",
"success","error","default".

<Container className="py-5 px-3 rounded-md my-6 bg-gray-100">
    <Text>Default</Text>
    <Input className="mb-2" variant="default" />
    <Text>Active</Text>
    <Input className="mb-2" variant="active" />
    <Text>Success</Text>
    <Input className="mb-2" variant="success" />
    <Text>Warning</Text>
    <Input className="mb-2" variant="warning" />
    <Text>Error</Text>
    <Input className="mb-2" variant="error" />
    <Text>Disabled</Text>
    <Input className="mb-2" variant="disabled" />
</Container>

#### Types

The "Input" comes with types, by default it uses the "text" variant, but you can also use these: "number", "text", "email", "currency", "date", "password",
"year", "month", "percentage", "file".

<Container className="py-5 px-3 rounded-md my-6 bg-gray-100">
    <Input type="text" />
</Container>

<WindowEditor codeString="import { Input } from 'dd360-ds'

<Input type='text' />" />

#### Disabled

For disabled Inputs, use the "disabled" prop.

</Container>

#### Large

For expansive size inputs, use the "large" prop.

<Container className="py-5 px-3 rounded-md my-6 bg-gray-100">

</Container>

#### boxShadow

Change boxShadow input, use the "boxShadow" prop with your variants: "sm", "base", "md", "lg", "xl", "2xl", "inner", "none".

<Container className="py-5 px-3 rounded-md my-6 bg-gray-100">
    <Input boxShadow="inner" />
</Container>

<Input boxShadow='inner' />" />
