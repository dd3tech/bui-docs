# TextArea

    The TextArea component is a React component that allows the user to input multiple lines of text

### Import

<WindowEditor codeString="import { TextArea } from 'dd360-ds'" />
<WindowEditor codeString="import TextArea from 'dd360-ds/TextArea'" />

### Usage

<ContainerComponentDoc className="gap-y-2">
    <TextArea />
</ContainerComponentDoc>

<WindowEditor codeString="import { TextArea } from 'dd360-ds'

<TextArea />" />

#### Variants

You can use the following variants: "default", "active", "success", "warning", "error" and "disabled". By default "active" is used.

<ContainerComponentDoc className="gap-y-2">
    <Text>Default</Text>
    <TextArea className="mb-2" variant="default" />
    <Text>Active</Text>
    <TextArea className="mb-2" variant="active" />
    <Text>Success</Text>
    <TextArea className="mb-2" variant="success" />
    <Text>Warning</Text>
    <TextArea className="mb-2" variant="warning" />
    <Text>Error</Text>
    <TextArea className="mb-2" variant="error" />
    <Text>Disabled</Text>
    <TextArea className="mb-2" variant="disabled" />
</ContainerComponentDoc>

<WindowEditor codeString="import { TextArea } from 'dd360-ds'

<TextArea variant='default' />
<TextArea variant='active' />
<TextArea variant='success' />
<TextArea variant='warning' />
<TextArea variant='error' />
<TextArea variant='disabled' />" />

#### Placeholder

Use the prop "placeholder" to indicate a help to the user about what information to enter in your TextArea component.

<ContainerComponentDoc className="gap-y-2">
    <TextArea placeholder="I'm a placeholder!" />
</ContainerComponentDoc>

<WindowEditor codeString="import { TextArea } from 'dd360-ds'

<TextArea placeholder='I am a placeholder!' />" />

#### Label

Use the prop "label" to indicate a help to the user about what information to enter in your TextArea component.

<ContainerComponentDoc className="gap-y-2">
    <TextArea label="I'm a label!" />
</ContainerComponentDoc>

<WindowEditor codeString="import { TextArea } from 'dd360-ds'

<TextArea label='I am a label!' />" />

#### TextArea With Icon

You can add an icon to the TextArea component by passing the prop startAdornment that receives a react component.

<ContainerComponentDoc className="gap-y-2">
    <TextArea endAdornment={<DynamicHeroIcon icon="AcademicCapIcon" className="text-blue-700 w-6" />} />
</ContainerComponentDoc>

<WindowEditor codeString="import { TextArea } from 'dd360-ds'
import DynamicHeroIcon from 'dd360-ds/DynamicHeroIcon'

<TextArea endAdornment={<DynamicHeroIcon icon='AcademicCapIcon' className='text-blue-700 w-6' />} />" />
