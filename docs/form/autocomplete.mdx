# <span name="floating-nav">AutoComplete</span>

    It is a useful component for creating a text input field that suggests auto-complete options to the user while typing.

##### <span name="floating-nav">Import</span>

<WindowEditor codeString="import { AutoComplete } from 'dd360-ds'" />
<WindowEditor codeString="import AutoComplete from 'dd360-ds/AutoComplete'" />

##### <span name="floating-nav">Usage</span>

It is mandatory to pass a prop called "items". 
The prop item, contains the data for each item in the drop-down list. Each array object must have a format similar to of following example:

<ContainerComponentDoc className="gap-y-2">
    <AutoComplete items={[{name: 'Item name', id: '1'}]} />
</ContainerComponentDoc>

<WindowEditor codeString="import { Checkbox } from 'dd360-ds'

<AutoComplete items={[{name: 'Item name', id: '1'}]} />" />

##### <span name="floating-nav">Variants</span>

Sets the appearance of the component. Available options are: default, active, disabled, warning, success, and error. Default uses: "default".

    <AutoComplete variant="warning" items={[]} />
    <Text className="mb-1">Success</Text>
    <AutoComplete variant="success" items={[]} />
    <Text className="mb-1">Error</Text>
    <AutoComplete variant="error" items={[]} />
    <Text className="mb-1">Disabled</Text>
    <AutoComplete variant="disabled" items={[]} />
</ContainerComponentDoc>

##### <span name="floating-nav">Types</span>

Sets the input type of the component. Available options are: text, number, email, currency, date, date, password, year, month, and percentage. By default it uses "text".

    <Text>Text</Text>
    <AutoComplete type="text" items={[]} />
    <Text>Number</Text>
    <AutoComplete type="number" items={[]} />
    <Text>Email</Text>
    <AutoComplete type="date" items={[]} />
    <Text>Password</Text>
    <AutoComplete type="password" items={[]} />
    <Text>Percentage</Text>
    <AutoComplete type="percentage" items={[]} />
</ContainerComponentDoc>

##### <span name="floating-nav">Can find text</span>

A message that is displayed when no items matching the user's search string are found.

<ContainerComponentDoc className="gap-y-2">
    <AutoComplete canFindText="No matches in your search" items={[]} />
</ContainerComponentDoc>

<WindowEditor codeString="import { Checkbox } from 'dd360-ds'

<AutoComplete canFindText='No matches in your search' items={[]} />" />

##### <span name="floating-nav">IsCloseOnBlur</span>

A boolean that indicates whether the drop-down list will be closed when the user clicks out of it.

<ContainerComponentDoc className="gap-y-2">
    <AutoComplete isCloseOnBlur={false} items={[]} />
</ContainerComponentDoc>

<WindowEditor codeString="import { Checkbox } from 'dd360-ds'

A boolean indicating whether the drop-down list is loading.
