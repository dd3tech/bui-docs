# <span name="floating-nav">Select</span>

   The component Select that renders a dropdown menu. The dropdown menu has several variants

### <span name="floating-nav">Import</span>

<WindowEditor codeString="import { Select } from 'dd360-ds'" />
<WindowEditor codeString="import Select from 'dd360-ds/Select'" />

### <span name="floating-nav">Usage</span>

To customize the options we must pass it an object in the prop optionsList with the structure of the example.

<Container className="py-5 px-3 rounded-md my-6 bg-gray-100">
    <Select optionsList={{ A: {label: 'Select option', disabled: true}, B: { label: 'Option B' }, C: { label: 'Option C' } }} />
</Container>

<WindowEditor codeString="import { Select } from 'dd360-ds'

<Select optionsList={{ A: {label: 'Select option', disabled: true}, B: { label: 'Option B' }, C: { label: 'Option C' } }} />" />

#### <span name="floating-nav">Variants</span>

Select has the following variants: "default", "active", "success", "warning", "error" and "disabled".
By default it uses "default".

#### <span name="floating-nav">Label</span>

The label prop is used to add a label to the Select component. When the prop is provided, the label will appear above the dropdown menu.

<Container className="py-5 px-3 rounded-md my-6 bg-gray-100">
    <Select
        label='I am a label'
        optionsList={{ A: {label: 'Select option', disabled: true}, B: { label: 'Option B' }, C: { label: 'Option C' } }}
    />
</Container>

<WindowEditor codeString="import { Select } from 'dd360-ds'

<Select
    &nbsp;&nbsp;label='I am a label'
    &nbsp;&nbsp;optionsList={{ A: {label: 'Select option', disabled: true}, B: { label: 'Option B' }, C: { label: 'Option C' } }}
/>
" />

#### <span name="floating-nav">noBorders</span>

The noBorders prop is used to remove the border styling of the Select component. When the prop is provided, the component will appear without borders. 

<Container className="py-5 px-3 rounded-md my-6 bg-gray-100">
    <Select
        noBorders
        optionsList={{ A: {label: 'Select option', disabled: true}, B: { label: 'Option B' }, C: { label: 'Option C' } }}
    />
</Container>

/>
" />
