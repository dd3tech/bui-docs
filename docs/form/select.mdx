# Select

   The component Select that renders a dropdown menu. The dropdown menu has several variants

### Import

<WindowEditor codeString="import { Select } from 'dd360-ds'" />
<WindowEditor codeString="import Select from 'dd360-ds/Select'" />

### Usage

To customize the options we must pass it an object in the prop optionsList with the structure of the example.

<Container className="py-5 px-3 rounded-md my-6 bg-gray-100">
    <Select optionsList={{ 0: {label: 'Select option', disabled: true}, A: { label: 'Option A' }, B: { label: 'Option B' } }} />
</Container>

<WindowEditor codeString="import { Select } from 'dd360-ds'

<Select optionsList={{ 0: {label: 'Select option', disabled: true}, A: { label: 'Option A' }, B: { label: 'Option B' } }} />" />

#### Variants

To customize the styles of the Select using the prop "variant", which contains the following options: "outline", "standard", and "none". 
By default it uses "outlined".

<Container className="py-5 px-3 rounded-md my-6 bg-gray-100">
    <Text className="block mb-1 mt-2">Standard</Text>
    <Select
        className="mb-3"
        variant="standard"
        optionsList={{ 0: {label: 'Select option', disabled: true}, A: { label: 'Option A' }, B: { label: 'Option B' } }}
    />
    <Text className="block mb-1 mt-2">Outlined</Text>
    <Select
        className="mb-3"
        variant="outlined"
        optionsList={{ 0: {label: 'Select option', disabled: true}, A: { label: 'Option A' }, B: { label: 'Option B' } }}
    />
    <Text className="block mb-1 mt-2">None</Text>
    <Select className="mb-3" variant="none" optionsList={{ 0: {label: 'Select option', disabled: true}, A: { label: 'Option A' }, B: { label: 'Option B' } }} />
</Container>

Select has the following states: "active", "success", "warning", "error" and "disabled".
By default it uses "active".
