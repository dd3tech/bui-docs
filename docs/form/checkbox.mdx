# <span name="floating-nav">Checkbox</span>

    This component is used to create a check box. The checkbox is a control that allows the user to select one or more options from a list.

##### <span name="floating-nav">Import</span>

<WindowEditor codeString="import { Checkbox } from 'dd360-ds'" />
<WindowEditor codeString="import Checkbox from 'dd360-ds/Checkbox'" />

### Usage

<Playground className="gap-y-2">
    <Checkbox />
</Playground>

<WindowEditor codeString="import { Checkbox } from 'dd360-ds'

<Checkbox />" />

##### <span name="floating-nav">Checked</span>

 This prop is used to set the initial state of the checkbox. If set to true, the checkbox will be checked. If set to false or not provided, the checkbox will appear unchecked.

<Playground className="gap-y-2">
    <Checkbox checked />
</Playground>

<WindowEditor codeString="import { Checkbox } from 'dd360-ds'

<Checkbox checked />" />

##### <span name="floating-nav">Indeterminate</span>

This property is used to set the indeterminate state of the checkbox. If set to true, the checkbox will appear in an intermediate state between checked and unchecked.

<Playground className="gap-y-2">
    <Checkbox indeterminate />
</Playground>

<WindowEditor codeString="import { Checkbox } from 'dd360-ds'

<Checkbox indeterminate />" />

##### <span name="floating-nav">Color</span>

This prop is used to set the color of the checkbox. Any valid CSS color value can be set.

<Playground className="gap-y-2">
    <Checkbox color="#000" />
</Playground>

<WindowEditor codeString="import { Checkbox } from 'dd360-ds'

<Checkbox color='#000' />" />

##### <span name="floating-nav">With control</span>

You can use the FormControlLabel component to add a label to the checkbox and control its status

<Playground className="gap-y-2">
    <FormControlLabel control={<Checkbox />} disabled label="Example" labelPlacement="top" />
</Playground>

<WindowEditor codeString="import { FormControlLabel } from 'dd360-ds'

<FormControlLabel control={<Checkbox />} label='Example' labelPlacement='top' />" />
