# <span name="floating-nav">FormControl</span>

    The FormControl component is used to provide a label to a form control, such as an input or checkbox.

##### <span name="floating-nav">Import</span>

<WindowEditor codeString="import { FormControlLabel } from 'dd360-ds'" />
<WindowEditor codeString="import FormControlLabel from 'dd360-ds/FormControlLabel'" />

##### <span name="floating-nav">Usage</span>

The prop label is required and specifies the text to be displayed next to the form control. 
The control property is also required and must contain the form control component.

<Playground className="align-center justify-center items-center">
    <FormControlLabel label="form-control" control={<Checkbox checked size='2xl' />} />
</Playground>

<WindowEditor codeString="import { FormControlLabel } from 'dd360-ds'

<FormControlLabel label='form-control' control={<Checkbox checked size='2xl' />} />" />

##### <span name="floating-nav">FormControl labelPlacement</span>

You can use the labelPlacement property to specify the position of the label relative to the control

<Playground className="align-center">
    <FormControlLabel labelPlacement="end" label="form-control" control={<Checkbox checked size='2xl' />} />
</Playground>

<WindowEditor codeString="import { FormControlLabel } from 'dd360-ds'

<FormControlLabel labelPlacement='end' label='form-control' control={<Checkbox checked size='2xl' />} />" />

##### <span name="floating-nav">FormControl disabled</span>

The disabled prop can be used to disable the form control

<Playground className="align-center">
    <FormControlLabel label="form-control" control={<Checkbox checked size='2xl' />} />
</Playground>
