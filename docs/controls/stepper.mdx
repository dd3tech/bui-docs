# <span name="floating-nav">Stepper</span>

With the Stepper component, you can display a circular progress bar indicating the status of a sequence of steps. You can also customize its appearance and it
provides a clear interface for users to track the progress of a task.

<br />
<br />
## <span name="floating-nav">Imports</span>

The first alternative is recommended since they can reduce the application bundle

<WindowEditor codeString="import Stepper from 'dd360-ds/Stepper'" />
<WindowEditor codeString="import { Stepper } from 'dd360-ds'" />

<br />
