# <span name="floating-nav">Spinner</span>

The Spinner component allows you to display the progress of a task or process in real time. It is visually appealing and provides instant feedback to the user
on the progress of the task, which improves the user experience on web or mobile applications. Plus, it's easy to use and customize.

<br />
<br />
##### <span name="floating-nav">Imports</span>

The first alternative is recommended since they can reduce the application bundle

<WindowEditor codeString="import Spinner from 'dd360-ds/Spinner'" />
<WindowEditor codeString="import { Spinner } from 'dd360-ds'" />

##### <span name="floating-nav">Color</span>

With the prop <Tag text='color' /> you can customize the color of the spinner

<Playground className="gap-y-2">
    <Spinner color="#44ce41" />
</Playground>

<WindowEditor
    codeString={`import  Progressbar from 'dd360-ds/Progressbar'

<Spinner color="#44ce41" />
`}

/>

<br />

##### <span name="floating-nav">Border</span>

With the prop <Tag text='border' /> you can customize the width of the spinner border

<Playground className="gap-y-2">
    <Spinner border={8} />
</Playground>

<WindowEditor
    codeString={`import  Progressbar from 'dd360-ds/Progressbar'

`}

/>

<br />

##### <span name="floating-nav">PageLoader</span>

With the prop <Tag text='pageLoader' /> you can add a proper spineer for page load

    <Spinner pageLoader />
</Playground>

    codeString={`import  Progressbar from 'dd360-ds/Progressbar'

`}

<br />

##### <span name="floating-nav">Width and Height</span>

With the props <Tag text='width' /> and <Tag text='height' />
you can customize the size of the spinner
