# <span name="floating-nav">Progressbar</span>

With the Progressbar component, you can display the progress of a task or process in real time.

<br />
<br />
## <span name="floating-nav">Imports</span>

The first alternative is recommended since they can reduce the application bundle

<WindowEditor codeString="import Progressbar from 'dd360-ds/Progressbar'" />
<WindowEditor codeString="import { Progressbar } from 'dd360-ds'" />

<br />
## <span name="floating-nav">Usage</span>

<ContainerComponentDoc className="gap-y-2">
    <ProgressBar value={20} height="10px" />
</ContainerComponentDoc>

<WindowEditor
    codeString={`import  Progressbar from 'dd360-ds/Progressbar'
  
<ProgressBar value={20} height="10px" />`}

/>

<br />

#### <span name="floating-nav">BackgroundColor</span>

With the prop <span className='border border-gray-300 bg-gray-200 p-1'>backgroundColor</span> you can customize the color of the bar

<ContainerComponentDoc className="gap-y-2">
    <ProgressBar backgroundColor="#93e78e" value={40} height="10px" />
</ContainerComponentDoc>

<WindowEditor
    codeString={`import  Progressbar from 'dd360-ds/Progressbar'
    
<ProgressBar backgroundColor="#93e78e" value={40} height="10px" />`}
/>
<br />
#### <span name="floating-nav">Label</span>

With the prop <span className='border border-gray-300 bg-gray-200 p-1'>label</span> you can add a text bar

<ContainerComponentDoc className="gap-y-2">
    <ProgressBar label="Loading..." value={60} height="20px" />
</ContainerComponentDoc>

<WindowEditor
 codeString={`import  Progressbar from 'dd360-ds/Progressbar'

<ProgressBar label="Loading..." value={60} height="20px" />
`}

/>

<br />
#### <span name="floating-nav">Animated</span>

With the prop <span className='border border-gray-300 bg-gray-200 p-1'>animated</span> you can add an animation to the bar

<ContainerComponentDoc className="gap-y-2">
    <ProgressBar animated value={80} height="10px" />
</ContainerComponentDoc>

<WindowEditor
  codeString={`import  Progressbar from 'dd360-ds/Progressbar'

<ProgressBar animated value={80} height="10px" />

`} />

<br />
#### <span name="floating-nav">BgColorContainer</span>

With the prop <span className='border border-gray-300 bg-gray-200 p-1'>animated</span> you can modify the color of the container of the bar

<ContainerComponentDoc className="gap-y-2">
    <ProgressBar backgroundColor="#ff4e4e" bgColorContainer="#fff" value={50} height="10px" />
</ContainerComponentDoc>

<WindowEditor
 codeString={`import  Progressbar from 'dd360-ds/Progressbar'

<ProgressBar backgroundColor="#ff4e4e" bgColorContainer="#fff" value={50} height="10px" />

`} />
