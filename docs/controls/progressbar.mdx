# <HeaderDocCustom title="Progressbar" url="https://main--62ffec7466615c40c8dbe435.chromatic.com/?path=/story/controls-progressbar--progress-bar" />


The ProgressBar component allows you to display the progress of a task or process in real time. The progress bar is a visual tool that indicates the percentage
of completion of a task, giving the user instant feedback on the progress of the process. The ProgressBar component is highly customizable, allowing you to
tailor it to the specific needs of your application. In addition, it is easy to use and its implementation significantly improves the user experience in your
web or mobile applications. With the ProgressBar component, you can make tracking the progress of a task simple and clear to users.

##### <span name="floating-nav">BackgroundColor</span>

With the prop <Tag text='backgroundColor' /> you can customize the color of the bar

<Playground className="gap-y-2">
    <ProgressBar backgroundColor="#93e78e" value={40} height="10px" />
</Playground>

<WindowEditor
    codeString={`import  Progressbar from 'dd360-ds/Progressbar'
    
<ProgressBar backgroundColor="#93e78e" value={40} height="10px" />
`}
/>
<br />
##### <span name="floating-nav">Label</span>

With the prop <Tag text='label' /> you can add a text bar

<Playground className="gap-y-2">
    <ProgressBar label="Loading..." value={60} height="20px" />
</Playground>

<WindowEditor
 codeString={`import  Progressbar from 'dd360-ds/Progressbar'

<ProgressBar label="Loading..." value={60} height="20px" />
`}

/>

<br />
##### <span name="floating-nav">Animated</span>

With the prop <Tag text='animated' /> you can add an animation to the bar

<Playground className="gap-y-2">
    <ProgressBar animated value={80} height="10px" />
</Playground>

<WindowEditor
  codeString={`import  Progressbar from 'dd360-ds/Progressbar'

##### <span name="floating-nav">BgColorContainer</span>

With the prop <Tag text='animated' /> you can modify the color of the container of the bar
