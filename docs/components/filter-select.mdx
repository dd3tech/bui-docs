# <HeaderDocCustom title="Filter Select" pathUrl="components-filterselect--filter-select" />


The FilterSelect component is used to create an interactive dropdown menu that allows the user to select an option from several predefined options in the application.

<br />
<br />
##### <span name="floating-nav">Imports</span>

<WindowEditor codeString="import { FilterSelect } from 'dd360-ds'" />

<br />
##### <span name="floating-nav">Usage</span>

Use the FilterSelect component as shown below:

<Playground className="gap-y-2">
 <FilterSelectCustom />
</Playground>

The code snippet below shows how to use the FilterSelect component:

<WindowEditor
    codeString={`import { FilterSelect } from 'dd360-ds'
  
<FilterSelect
    listItems={{
      A: {
        label: 'Option A'
      },
      B: {
        label: 'Option B'
      }
    }}
    onApply={handleChange}
/>
`}
/>
