# <span name="floating-nav">DatePicker</span>

With the DatePicker component, you can provide users with the ability to interactively select dates in web applications. It allows easy entry and manipulation of temporal data, facilitating the selection and presentation of dates in a friendly and customizable format.

<br />

## <span name="floating-nav">Imports</span>

The first alternative is recommended since they can reduce the application bundle

<WindowEditor codeString="import DatePicker from 'dd360-ds/DatePicker'" />
<WindowEditor codeString="import { DatePicker } from 'dd360-ds'" />

## <span name="floating-nav">Usage</span>

Use the DatePicker component as shown below:

<ContainerComponentDoc>
  <DatePicker className="date-picker" language="en" />
</ContainerComponentDoc>
The code snippet below shows how to use the DatePicker component:

<WindowEditor
    codeString={`import  DatePicker  from 'dd360-ds/DatePicker'

<DatePicker />
`}/>

#### <span name="floating-nav">Language</span>

With the prop <span className="border border-gray-300 bg-gray-200 p-1">language</span>, its variants <span className="border border-gray-300 bg-gray-200 p-1">es</span> and <span className="border border-gray-300 bg-gray-200 p-1">en</span> you can control the language

<ContainerComponentDoc>

<Flex gap='12'>
  <DatePicker className="date-picker" language="en" />
  <DatePicker className="date-picker" language="es" />
</Flex>
</ContainerComponentDoc>

<WindowEditor
  codeString={`import { Flex, DatePicker } from 'dd360-ds''
  
<Flex gap='12'>
  <DatePicker language="en" />
  <DatePicker language="es" />
</Flex>
`}
/>

#### <span name="floating-nav">Format</span>

With the prop <span className="border border-gray-300 bg-gray-200 p-1">format</span>, and its <span className="border border-gray-300 bg-gray-200 p-1">short</span> and <span className="border border-gray-300 bg-gray-200 p-1">long</span> variants, you can customize the display format of the date, either in its full or abbreviated form.

<ContainerComponentDoc>
<Flex justifyContent='around'>
  <DatePicker className="date-picker" format="short" language='en' />
  <DatePicker className="date-picker" format="long"  language='en' />
</Flex>
</ContainerComponentDoc>

<WindowEditor
  codeString={`import  DatePicker from 'dd360-ds/DatePicker''
  
 <Flex justifyContent='around'>
  <DatePicker format="short" language='en' />
  <DatePicker format="long"  language='en' />
</Flex>
`}
/>

#### <span name="floating-nav">OnlyOf</span>

With the prop <span className="border border-gray-300 bg-gray-200 p-1">onlyOf</span>, its variants <span className="border border-gray-300 bg-gray-200 p-1">year</span>, <span className="border border-gray-300 bg-gray-200 p-1">month</span>, <span className="border border-gray-300 bg-gray-200 p-1">day</span> and <span className="border border-gray-300 bg-gray-200 p-1">month-year</span> you can control the format

<ContainerComponentDoc>

<Flex gap="12" justifyContent="around">
  <DatePicker className="date-picker" language="en" onlyOf="year" />
  <DatePicker className="date-picker" language="en" onlyOf="month" />
  <DatePicker className="date-picker" language="en" onlyOf="day" />
  <DatePicker className="date-picker" language="en" onlyOf="month-year" />
</Flex>

</ContainerComponentDoc>

<WindowEditor
  codeString={`import { Flex, DatePicker } from 'dd360-ds''
  
<Flex gap="12" justifyContent="around">
  <DatePicker language="en" onlyOf="year" />
  <DatePicker language="en" onlyOf="month" />
  <DatePicker language="en" onlyOf="day" />
  <DatePicker language="en" onlyOf="month-year" />
</Flex>
`} />

#### <span name="floating-nav">MinDate</span>

With the prop <span className="border border-gray-300 bg-gray-200 p-1">minDate</span>, you can define the selectable minimum date

<ContainerComponentDoc>

<DatePicker className="date-picker" language="en" value={new Date('2023-05-16T03:00:00.000Z')} minDate={new Date('2023-05-10T03:00:00.000Z')} />

</ContainerComponentDoc>

<WindowEditor
  codeString={`import  DatePicker  from 'dd360-ds/DatePicker''

`}
/>

With the prop <span className="border border-gray-300 bg-gray-200 p-1">maxDate</span>, you can define the maximum selectable date
