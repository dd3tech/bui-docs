# <span name="floating-nav">Timeline</span>

The Timeline component is a visual representation of a timeline where events or
content can be displayed in chronological order. It allows displaying
information sequentially and facilitates the visualization of the progression of
events over time.

<br />

## <span name="floating-nav">Imports</span>

You can import the Timeline component in the following ways. The first
alternative is recommended since they can reduce the application bundle

<WindowEditor codeString="import Timeline from 'dd360-ds/Timeline'" />
<WindowEditor codeString="import { Timeline } from 'dd360-ds'" />

## <span name="floating-nav">Usage</span>

With the Timeline component, you can easily create a visually appealing timeline to showcase a series of events or steps in a user-friendly manner. Each timeline item can be customized with various content and styling options to suit your specific needs.
