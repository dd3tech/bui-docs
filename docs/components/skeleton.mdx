# <span name="floating-nav">Skeleton</span>

The Skeleton component in React is a tool to display a skeleton while loading dynamic content. Provides a rough visual representation of the structure of the elements to be loaded, improving the user
experience. It is commonly used in situations where slow loading of content is expected.

<br />

##### <span name="floating-nav">Table Content Skeleton</span>

With the component <Tag text=' TableContentSkeleton' /> you can display a skeleton table content with mock rows and columns to represent data in a
tabular structure.

<ContainerComponentDoc>
  <Table>
    <Table.Header>
      <Table.HeaderRow>
        <Table.HeaderCell className="bg-gray-100">Col 1</Table.HeaderCell>
        <Table.HeaderCell className="bg-gray-100">Col 2</Table.HeaderCell>
        <Table.HeaderCell className="bg-gray-100">Col 3</Table.HeaderCell>
        <Table.HeaderCell className="bg-gray-100">Col 4</Table.HeaderCell>
      </Table.HeaderRow>
    </Table.Header>
    <Table.Body>
      <TableContentSkeleton rows={10} columns={4} rounded="full" skeletonHeight={10} classNameCell="px-4 border" skeletonWidth={[100, 100, 75, 40]} skeletonClass="m-auto" skeletonColor="#e5e7eb" />
    </Table.Body>
  </Table>
</ContainerComponentDoc>
