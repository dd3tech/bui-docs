# <span name="floating-nav">BarList</span>


The BarList component is a customizable React component designed to display a list of bars representing different data points. 
It provides a visually appealing way to present metrics, values, or progress indicators in a clear and intuitive manner.

<br />

## <span name="floating-nav">Imports</span>
The first alternative is recommended since it can reduce the application bundle

<WindowEditor codeString="import BarList from 'dd360-ds/BarList'" />
<WindowEditor codeString="import { BarList } from 'dd360-ds'" />

<br />

Use the BarList component as shown below:

<ContainerComponentDoc>
    <BarList
