# <span name="floating-nav">BarList</span>

The BarList component is a customizable React component that allows you to create 
ordered or unordered lists with various styling options.

<br />

## <span name="floating-nav">Imports</span>

<WindowEditor codeString="import { BarList } from 'dd360-ds'" />

## <span name="floating-nav">Usage</span>

The BarList component in React allows you to create and customize lists with ease. Whether 
you need an ordered or unordered list, you can easily specify the type and customize the 
appearance using various props. Adjust the gap between items, style the icons, and add 
labels to enhance the list's visual appeal. You can even include custom icons for each 
list item using the BarList.Icon component. Simply populate the list by using the BarList.Item 
component as children. The BarList component offers a straightforward and flexible way to create 
functional and visually appealing lists in your React applications.

Here's an example of how you can use the BarList component to create:

<ContainerComponentDoc>
  <BarList dataList={[{ label: 'Snapchat', value: 0 },
    { label: 'Linkedin', value: 25 },
    { label: 'Twitter', value: 50 },
    { label: 'Facebook', value: 75 },
    { label: 'Instagram', value: 100 }]} />
</ContainerComponentDoc>
