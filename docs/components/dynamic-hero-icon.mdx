# <HeaderDocCustom title="DynamicHeroIcon" pathUrl="components-dynamicheroicon--dynamic-hero-icon" />

The DynamicHeroIcon component allows you to display dynamic icons, adding an attractive and custom visual layer in a faster way.

<br />
<br />
##### <span name="floating-nav">Imports</span>

<WindowEditor codeString="import DynamicHeroIcon from 'dd360-ds/DynamicHeroIcon'" />

<br />
##### <span name="floating-nav">Usage</span>

Use the DynamicHeroIcon component as shown below:

<Playground className="gap-y-2">
       <DynamicHeroIcon icon="HomeIcon" width={30} className="text-blue-700" />
</Playground>

The code snippet below shows how to use the DynamicHeroIcon component:

<WindowEditor
    codeString={`import DynamicHeroIcon from 'dd360-ds/DynamicHeroIcon'
  
<DynamicHeroIcon icon="HomeIcon" width={30} className="text-blue-700" />`}
/>

##### <span name="floating-nav">Icon</span>

With the prop <TagDocs text='Icon' />, you can choose the required icon in a fast and simple way

<Playground className="gap-y-2">
       <DynamicHeroIcon icon="BellIcon" width={30} className="text-blue-700" />
</Playground>


<WindowEditor
    codeString={`import DynamicHeroIcon from 'dd360-ds/DynamicHeroIcon'
  
<DynamicHeroIcon icon="BellIcon" width={30} className="text-blue-700" />`}
/>

##### <span name="floating-nav">Width</span>

With the prop <TagDocs text='width' />, you can choose the size of the chosen icon

<Playground className="gap-y-2">
       <DynamicHeroIcon icon="BellIcon" width={50} className="text-blue-700" />
</Playground>


<WindowEditor
    codeString={`import DynamicHeroIcon from 'dd360-ds/DynamicHeroIcon'
  
<DynamicHeroIcon icon="BellIcon" width={50} className="text-blue-700" />`}
/>

##### <span name="floating-nav">ClassName</span>

With the prop <TagDocs text='className' />, you can choose the color of the chosen icon, among many other things

<Playground className="gap-y-2">
       <DynamicHeroIcon icon="CheckCircleIcon" width={30} className="text-green-400" />
</Playground>


<WindowEditor
    codeString={`import DynamicHeroIcon from 'dd360-ds/DynamicHeroIcon'
  
<DynamicHeroIcon icon="CheckCircleIcon" width={30} className="text-green-400" />`}
/>
