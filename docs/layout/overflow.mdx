# <span name="floating-nav">Overflow</span>

The Overflow component allows you to control the display of the content inside a container when it overflows the visible area.

<br />
<br />
##### <span name="floating-nav">Import</span>

<WindowEditor codeString="import { Overflow } from 'dd360-ds'" />
<WindowEditor codeString="import Overflow from 'dd360-ds/Layout/Overflow'" />

<br />
<br />
##### <span name="floating-nav">Usage</span>

The prop "overflow" indicates how the overflow of the content inside the container is controlled. Possible values are 'hidden', 'auto', 'visible', 'x-auto',
'x-hidden', 'x-visible', 'scroll', 'y-auto', 'y-hidden', 'y-visible', 'x-scroll' and 'y-scroll'. The default value is 'auto'.
