# Download Card

    It is a download card that allows the user to download a file. 

### Import

<WindowEditor codeString="import { DownloadCard } from 'dd360-ds'" />
<WindowEditor codeString="import DownloadCard from 'dd360-ds/DownloadCard'" />

### Usage

The following 3 props are required to use the component

- cancelText: Text to be displayed in the cancel button.
- downloadText: Text displayed on the download button.
- description: Description of the content to be downloaded.

<ContainerComponentDoc>
    <DownloadCard
        cancelText="Do not remind me again"
        downloadText="Download Template"
        description="Download the excel template, fill in the necessary fields and re-upload the file to populate the table. You can also fill the table manually."
    />
</ContainerComponentDoc>

#### isActive

If true, the card is shown; if false, the card is hidden. Default is true

        isActive={false}
    />
</ContainerComponentDoc>
