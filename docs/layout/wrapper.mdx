# Wrapper

    The Wrapper component is a container used to wrap other components and provide them with a common set of styles.

### Import

<WindowEditor codeString="import { Wrapper } from 'dd360-ds'" />
<WindowEditor codeString="import Wrapper from 'dd360-ds/Wrapper'" />

### Usage

<ContainerComponentDoc>
   <Wrapper className="bg-blue-500">I am a children</Wrapper>
</ContainerComponentDoc>

<WindowEditor codeString="import { Wrapper } from 'dd360-ds'

<Wrapper className='bg-blue-500' >I am a children</Wrapper>" />

#### hasViewportHeight

Sets whether the component should have a viewport height. 
The default value is "false".

<ContainerComponentDoc>
   <Wrapper className='bg-blue-500' hasViewportHeight >I am a children</Wrapper>
</ContainerComponentDoc>

<WindowEditor codeString="import { Wrapper } from 'dd360-ds'

<Wrapper hasViewportHeight className='bg-blue-500' >I am a children</Wrapper>" />

#### maxWidth

Defines the maximum width of the component, and can take one of the following values: "max", "min", "none", "0", "sm", "md", "lg", "xl", "2xl", "3xl", "full", "4xl", "5xl", "xs", "7xl", "6xl", "prose", "screen-sm", "screen-md", "screen-lg", "screen-xl", "screen-2xl". 
By default, it is set to "screen-2xl".

<ContainerComponentDoc>
   <Wrapper  maxWidth='none' className='bg-blue-500 mb-2'>none</Wrapper>
   <Wrapper  maxWidth='sm' className='bg-blue-500 mb-2'>sm</Wrapper>
   <Wrapper  maxWidth='md' className='bg-blue-500 mb-2'>md</Wrapper>
   <Wrapper  maxWidth='full' className='bg-blue-500 mb-2'>full</Wrapper>
   <Wrapper  maxWidth='min' className='bg-blue-500 mb-2'>min</Wrapper>
   <Wrapper  maxWidth='max' className='bg-blue-500 mb-2'>max</Wrapper>
   <Wrapper  maxWidth='screen-sm' className='bg-blue-500 mb-2'>screen-sm</Wrapper>
   <Wrapper  maxWidth='screen-md' className='bg-blue-500 mb-2'>screen-md</Wrapper>
</ContainerComponentDoc>
