##### <span name="floating-nav">HasViewportHeight</span>

Sets whether the component should have a viewport height. The default value is "false".

##### <span name="floating-nav">MaxWidth</span>

Defines the maximum width of the component, and can take one of the following values: "max", "min", "none", "0", "sm", "md", "lg", "xl", "2xl", "3xl", "full",
"4xl", "5xl", "xs", "7xl", "6xl", "prose", "screen-sm", "screen-md", "screen-lg", "screen-xl", "screen-2xl". By default, it is set to "screen-2xl".
