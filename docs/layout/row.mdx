# <HeaderDocCustom title="Row" url="https://main--62ffec7466615c40c8dbe435.chromatic.com/?path=/story/layout-row--row" />


    The Row component is a flexible container for arranging items in a single row, using a grid system that allows you to configure the width of columns for different screen sizes.
    It allows you to specify a number of columns and a spacing between them.

##### <span name="floating-nav">Custom Row</span>

You can customize the Row component with the following props:

"cols": number of columns to display in the row. Default is 4. Accepts values from 1 to 4. "md": number of columns to display in the row for medium sized
screens. Default is 2. Accepts values from 1 to 4. "sm": number of columns to show in the row for small screens. Default is 1. Accepts values from 1 to 4.
"gap": space between columns in the row. Default is 4. Accepts values from 1 to 5.

<Playground>
    <Row cols={3} md={2} sm={1}>
        <div>One div</div>
        <div>Two div</div>
        <div>Three div</div>
    </Row>
</Playground>
