# <span name="floating-nav">SideBar</span>

The SideBar component is a sidebar panel that displays a list of navigation items.
It is useful for creating user interfaces with hierarchical navigation, where users can access different sections
or pages within an application.

The sidebar consists of a header, a list of sidebar items, and an optional danger zone.
The header displays the name and subtitle of the sidebar. The sidebar items represent the navigation options
and can have dropdown sub-items. The danger zone provides an additional option with a warning icon for high-risk actions.

## <span name="floating-nav">Usage</span>

The SideBar component is highly customizable and adaptable to different use cases. It can show or hide items,
expand or collapse the sidebar, and adjust the style and position as needed.

<br />
#### <span name="floating-nav">sideBarName</span>
The prop sideBarName is required, it is a string that is rendered as the title of the SideBar component.

<WindowEditor codeString='import { Sidebar } from "dd360-ds"

<SideBar sideBarName="Sidebar" />' />

<br />
#### <span name="floating-nav">sideBarSubTitle</span>

A string that is rendered as the subtitle of the SideBar.

<WindowEditor codeString='import { Sidebar } from "dd360-ds"

<SideBar sideBarName="Sidebar" sideBarSubTitle="Example SubTitle" />' />

<br />
#### <span name="floating-nav">sideBarList</span>

A list of objects representing the sidebar options. Each object must have the same structure as shown in the following example:

<ul style={{listStyle: 'revert', padding: 'revert'}}>
    <li>title (string, required): The title of the sidebar item.</li>
    <li>active (boolean, required): Indicates if the sidebar item is active.</li>
    <li>isOpen (boolean): Indicates if the sub-menu of the sidebar item is open.</li>
    <li>goTo (function): The callback function to be executed when the sidebar item is selected.</li>
    <li>icon (JSX.Element): The icon to be displayed next to the title of the sidebar item.</li>
    <li>disabled (boolean): Indicates if the sidebar item is disabled.</li>
    <li>hidden (boolean): Indicates if the sidebar item is hidden.</li>
    <li>subItems (SideBarSubItem): An object representing the sub-items of the sidebar item. Each property of the object should have a unique key and an object with the following properties:</li>
    <ul style={{listStyle: 'revert', padding: 'revert'}}>
        <li>title (string, required): The title of the sub-item.</li>
        <li>active (boolean, required): Indicates if the sub-item is active.</li>
        <li>goTo (function): The callback function to be executed when the sub-item is selected.</li>
    </ul>
</ul>

<br />
#### <span name="floating-nav">dangerZone</span>

An object representing the danger zone of the SideBar. The object must have the same structure as shown in the following example:

<ul style={{listStyle: 'revert', padding: 'revert'}}>
    <li>show (boolean): Indicates if the danger zone is shown.</li>
    <li>text (string): The text to be displayed in the danger zone.</li>
    <li>active (boolean): Indicates if the danger zone is active.</li>
    <li>callBack (function): The callback function to be executed when the danger zone is selected.</li>
</ul>

Example:

<WindowEditor codeString='import { Sidebar } from "dd360-ds"

<SideBar
&nbsp;&nbsp;sideBarName="Sidebar"
&nbsp;&nbsp;dangerZone={{
&nbsp;&nbsp;&nbsp;&nbsp;active: false,
&nbsp;&nbsp;&nbsp;&nbsp;show: true,
&nbsp;&nbsp;&nbsp;&nbsp;text: "Delete project"
&nbsp;&nbsp;&nbsp;&nbsp;callBack: () => {}
&nbsp;&nbsp;}} 
/>
' />

<br />
#### <span name="floating-nav">disabledOptionsTag</span>

A string representing the text to be displayed in the disabled options.

<WindowEditor codeString='import { Sidebar } from "dd360-ds"

<SideBar sideBarName="Sidebar" disabledOptionsTag="Coming soon" />' />

<br />
#### <span name="floating-nav">isLoadingHeaderInfo</span>

A boolean which indicates when the header information is loading, while it is true Skeletons will be shown.

<WindowEditor codeString='import { Sidebar } from "dd360-ds"

<SideBar sideBarName="Sidebar" IsLoadingHeaderInfo={true} />' />

<br />

#### <span name="floating-nav">isLoadingSideBarList</span>

A boolean indicating when the option listing information is being loaded, as long as it is true, the skeletons will be displayed.

<WindowEditor codeString='import { Sidebar } from "dd360-ds"

<SideBar sideBarName="Sidebar" isLoadingSideBarList={true} />' />

<br />

#### <span name="floating-nav">numSkeletons</span>

Number of items of type skeletons to show when isLoadingSideBarList is true.

<WindowEditor codeString='import { Sidebar } from "dd360-ds"

<SideBar sideBarName="Sidebar" isLoadingSideBarList={true} numSkeletons={5} />' />


<br />
#### <span name="floating-nav">defaultExpand</span>

A boolean representing whether the sidebar should be expanded by default.

<WindowEditor codeString='import { Sidebar } from "dd360-ds"

<SideBar sideBarName="Sidebar" defaultExpand />' />


<br />
