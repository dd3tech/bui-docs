##### <span name="floating-nav">Shadow</span>

Used to apply different shadow styles to the container. Options include none, sm, md, lg, xl, and inner.

<ContainerComponentDoc>
    <Container className="mb-2" shadow="none">

    </Container>
    <Container className="mb-2" shadow="sm">
        sm
    </Container>
    <Container className="mb-2" shadow="md">
        md
    </Container>
    <Container className="mb-2" shadow="lg">
        lg
    </Container>
    <Container className="mb-2" shadow="xl">
        xl
    </Container>
    <Container className="mb-2" shadow="inner">
        inner
    </Container>
</ContainerComponentDoc>
