# <HeaderDocCustom title="TopPage" pathUrl="components-toppage" />

The TopPage component is designed to create a hierarchical page structure with navigation elements, ideal for complex applications that require organized sections.

This component is especially useful in management applications, dashboards, or any system that needs to organize multiple sections and important actions in a compact 
 and easy-to-navigate interface.

##### <span name="floating-nav">Usage</span>

The TopPage component is highly versatile and suited for a variety of applications. It allows for the display of breadcrumb navigation, configurable action buttons, 
and a customizable header. 

Users can easily adjust the layout, add tabs for organized sections, and include icons or last-update timestamps to enhance usability and structure.

##### <span name="floating-nav">OptionsBreadcrumbs</span>

The optionsBreadcrumbs prop is an optional array that defines the breadcrumb navigation for the TopPage component,
allowing users to see their navigation path and easily return to previous sections. Each breadcrumb item includes a label and link.

<Playground className="gap-y-2">
   <TopPage title="Title Top Page" children={<div>Children Example</div> } optionsBreadcrumbs={[{ name: "Home", to: () => alert("Action Home") },{ name: "Category", to: () => alert("Action Category") }]} />
</Playground>

##### <span name="floating-nav">LastUpdate</span>

The lastUpdate prop is an optional object that displays the date of the last update on the TopPage component. 
It includes a translation key to set the language ("en" or "es") and a date key to specify the date, helping users stay informed about recent changes or updates.

<Playground className="gap-y-2">
   <TopPage title="Title Top Page" children={<div>Children Example</div> }  optionsBreadcrumbs={[{ name: "Home", to: () => alert("Action Home") },{ name: "Category", to: () => alert("Action Category") }]} lastUpdate={{ translation: "es", date: new Date()}} />
</Playground>

##### <span name="floating-nav">CallToActionsButtons</span>

The callToActionsButtons prop is an optional array of action buttons displayed in the header of the TopPage component. 
Each button object can include a label, icon, style variant, and click handler, providing quick access to key actions for the user. 
Up to four buttons can be displayed.

<Playground className="gap-y-2">
   <TopPage title="Title Top Page" 
   children={<div>Children Example</div> }  
   optionsBreadcrumbs={[{ name: "Home", to: () => alert("Action Home") },{ name: "Category", to: () => alert("Action Category") }]} lastUpdate={{ translation: "es", date: new Date()}}
   callToActionsButtons={  [
    {
      label: "Action 1",
      onClick: () => alert("Action 1 clicked"),
      variant: "primary",
      isDisabled: true
    }]}
    />
</Playground>
<br/>

##### <span name="floating-nav">ActionIcon</span>

The actionIcon prop is an optional object that represents a standalone icon button in the header of the TopPage component. 
It includes properties for an icon, click handler, and states for selection and disabling, allowing users to perform a key action independently of the main action buttons.

<WindowEditor codeString='import { TopPage } from "dd360-ds"

<TopPage title="Title Top Page" children={<div>Children Example</div>} 
actionIcon={
    titleIcon: <CheckCircleIcon />,
    onClick: () => alert("Icon clicked"),
    isSelected: false,
    isDisabled: true
  } 
  />' />

<br />

##### <span name="floating-nav">Tabs</span>

The tabs prop is an optional object that configures tabbed navigation within the TopPage component.
It includes properties for the current tab value, a setValue function to update the active tab, and an items array defining each tab with a label and optional disabled state.
This allows users to switch between organized sections of content seamlessly.

<Playground className="gap-y-2">
   <TopPage title="Title Top Page"
            children={<div>Children Example</div>}
            tabs={{
                  value: 0,
                  setValue: (value) => alert("Tab changed to", value),
                  items: [
                      { label: "Tab 1" },{ label: "Tab 2", disabled:true }
                  ]
              }} 
    />
</Playground>
