# <HeaderDocCustom title="Pagination" url="https://main--62ffec7466615c40c8dbe435.chromatic.com/?path=/story/navigation-pagination--pagination" />


    The Pagination component is a handy tool that facilitates user navigation through different pages, especially when used with tables. With a minimalist interface and clear controls for moving pages forward or backward, jumping to a specific page and adjusting the number of elements displayed per page, this component is highly customizable and complies with accessibility standards. It improves usability and efficiency when handling large data sets, making it a valuable addition.

<br />
##### <span name="floating-nav">Imports</span>

The first alternative is recommended since they can reduce the application bundle.

<WindowEditor codeString="import Pagination from 'dd360-ds/Pagination'" />
<WindowEditor codeString="import { Pagination } from 'dd360-ds'" />

##### <span name="floating-nav">Usage</span>

You can use it by importing the <Tag text='usePagination' /> hook and use it like this

<Playground className="flex items-center gap-x-5">
    <PaginationCustom />
</Playground>

<WindowEditor
    codeString={`import { Pagination, usePagination } from 'dd360-ds'

const paginationProps = usePagination()

<Pagination {...paginationProps} firstText="Show" secondText="of the 50 items" currentPage={paginationProps.currentPage + 1} totalPages={10} />

`} />

Or you can customize it your way with your functions and variables

<WindowEditor
    codeString={`import Pagination from 'dd360-ds/Pagination'

<Pagination
    totalPages={5}
    currentPage={1}
    sliceSize="10"
    firstText="Show"
    secondText="of the 65 items"
    goToPreviousPage={() => alert('goToBackPage')}
    goToNextPage={() => alert('goToNextPage')}
    goToPage={() => alert('goPage')}
    setSize={() => null}
/>
`} />
