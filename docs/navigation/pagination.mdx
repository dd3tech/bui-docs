# <span name="floating-nav">Pagination</span>

    The Pagination component can make it easier for the user to navigate through the different pages.

<br />
## <span name="floating-nav">Imports</span>

<WindowEditor codeString="import { Pagination } from 'dd360-ds'" />

## <span name="floating-nav">Usage</span>

<ContainerComponentDoc className="flex items-center">
    <Pagination
        totalPages={5}
        currentPage={1}
        sliceSize="10"
        firstText="Show"
        secondText="of the 65 items"
        goToPreviousPage={() => alert('goToBackPage')}
        goToNextPage={() => alert('goToNextPage')}
        goToPage={() => alert('goPage')}
        setSize={() => null}
    />
</ContainerComponentDoc>
