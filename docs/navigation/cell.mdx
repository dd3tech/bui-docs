# <HeaderDocCustom title="Cell" pathUrl="navigation-cell--cell" />


With the Cell you can create an interactive cell that can have a button or an icon, along with some text. The cell can be enabled or disabled, selected or
unselected, and may or may not have a border. Cell has a link that can be clicked.
