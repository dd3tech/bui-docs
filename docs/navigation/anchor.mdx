# <span name="floating-nav">Anchor</span>

With the Anchor component you can create links with customizable properties and support for custom components.

<br />

## <span name="floating-nav">Imports</span>

The first alternative is recommended since they can reduce the application bundle.

<WindowEditor codeString="import Anchor from 'dd360-ds/Anchor'" />
<WindowEditor codeString="import { Anchor } from 'dd360-ds'" />

## <span name="floating-nav">Usage</span>

<ContainerComponentDoc className="flex items-center">
    <Anchor to="/docs/navigation/anchor">This is an example</Anchor>
</ContainerComponentDoc>

<WindowEditor
    codeString="import Anchor from 'dd360-ds/Anchor''

        <Anchor>This is an example</Anchor> "

/>

<br />

#### <span name="floating-nav">To</span>

With the prop <span className='border border-gray-300 bg-gray-200 p-1'>to</span> you can redirect to the url you need

<ContainerComponentDoc className="flex items-center">
    <Anchor to="/docs/navigation/anchor">This is an example with the prop to</Anchor>
</ContainerComponentDoc>

<WindowEditor
    codeString={`import Anchor from 'dd360-ds/Anchor'

<Anchor to="/docs/navigation/anchor">This is an example with the prop to</Anchor>
`}/>

<br />

#### <span name="floating-nav">As</span>

With the prop <span className='border border-gray-300 bg-gray-200  p-1'>as</span> you can add the Link of react router dom.

<ContainerComponentDoc className="flex items-center">
    <Anchor to="/docs/navigation/anchor">This is an example with the prop as</Anchor>
</ContainerComponentDoc>

<WindowEditor
    codeString={`import Anchor from 'dd360-ds/Anchor'
import { Link } from 'react-router-dom'
    
<Anchor as={Link} to="/docs/navigation/anchor">
    This is an example with the prop as
</Anchor>
`}
/>
