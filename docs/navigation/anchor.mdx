# <span name="floating-nav">Anchor</span>

With the Anchor component you can create links with customizable properties and support for custom components.

<br />

#### <span name="floating-nav">To</span>

With the props to you can redirect to the url you need

    <Anchor to={'/docs/navigation/anchor'}>This is an example with the props to</Anchor>
</ContainerComponentDoc>

    codeString={`import { Anchor } from 'dd360-ds'

<br />

#### <span name="floating-nav">LinkComponent</span>

With the LinkComponent prop you can add the Link of react router dom.

<ContainerComponentDoc className="flex items-center">
    <Anchor to={'/docs/navigation/anchor'}>This is an example with the props LinkComponent</Anchor>
</ContainerComponentDoc>

<WindowEditor
    codeString={`import { Anchor } from 'dd360-ds'
import { Link } from 'react-router-dom'
    
<Anchor LinkComponent={Link} to="/docs/navigation/anchor">
    This is an example with the props LinkComponent
</Anchor>
`}
/>
