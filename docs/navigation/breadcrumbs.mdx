# <span name="floating-nav">Breadcrumbs</span>

    The Breadcrumbs component displays the user's current location on the website using a breadcrumb path.

<br />

##### <span name="floating-nav">Imports</span>

The first alternative is recommended since they can reduce the application bundle.

<WindowEditor codeString="import Breadcrumbs from 'dd360-ds/Breadcrumbs'" />
<WindowEditor codeString="import { Breadcrumbs } from 'dd360-ds'" />

##### <span name="floating-nav">Usage</span>

<ContainerComponentDoc className="flex items-center relative">
    <Breadcrumbs options={[{ name: 'First Path' }, { name: 'Second Path' }, { name: 'Third Path' }]} />
</ContainerComponentDoc>

<WindowEditor
    codeString="import Breadcrumbs from 'dd360-ds/Breadcrumbs'

      <Breadcrumbs options={[{name: 'First Path'},{ name: 'Second Path'},{ name: 'Third Path'}]}/> "

/>

<br />

##### <span name="floating-nav">Options</span>

The prop option is an array of objects that contains information about the routes in the Breadcrumbs component, including the name, icon, and redirect function
of each route

<ContainerComponentDoc className="flex items-center relative">
    <Breadcrumbs
        options={[
            {
                name: 'Home',
                icon: () => <DynamicHeroIcon icon="HomeIcon" width="24" />,
                to: () => (window.location.href = '/')
            },
            {
                name: 'Docs',
                icon: () => <DynamicHeroIcon icon="BeakerIcon" width="24" />,
                to: () => (window.location.href = '/docs/get-started/getting-started')
            },
            {
                name: 'Breadcrumbs',
                icon: () => <DynamicHeroIcon icon="CalculatorIcon" width="24" />,
                to: () => (window.location.href = '/docs/navigation/breadcrumbs')
            }
        ]}
    />
</ContainerComponentDoc>

<WindowEditor
    codeString={`import Breadcrumbs from 'dd360-ds/Breadcrumbs'
import DynamicHeroIcon from 'dd360-ds/DynamicHeroIcon'

<Breadcrumbs
    options={[
        { name: 'Home', icon: () => <DynamicHeroIcon icon="HomeIcon" width="24" />, to: () => (window.location.href = '/') },
        {
            name: 'Docs',
            icon: () => <DynamicHeroIcon icon="BeakerIcon" width="24" />,
            to: () => (window.location.href = '/docs/get-started/getting-started')
        },
        {
            name: 'Breadcrumbs',
            icon: () => <DynamicHeroIcon icon="CalculatorIcon" width="24" />,
            to: () => (window.location.href = '/docs/navigation/breadcrumbs')
        }
    ]}
/>
`}/>

##### <span name="floating-nav">Separator</span>

With the prop <Tag text='separator' /> you can to change the default separator icon between the options

<ContainerComponentDoc className="flex items-center relative">
    <Breadcrumbs options={[{ name: 'First Path' }, { name: 'Second Path' }, { name: 'Third Path' }]} separator="|" />
</ContainerComponentDoc>

<WindowEditor
    codeString={`import Breadcrumbs from 'dd360-ds/Breadcrumbs'
import DynamicHeroIcon from 'dd360-ds/DynamicHeroIcon'

<Breadcrumbs options={[{ name: 'First Path' }, { name: 'Second Path' }, { name: 'Third Path' }]} separator="|" />
`} />
